Master thesis program for automated analysis of bone raman spectra.

Python3 <= is required. 
Required external packages (download via pip):
- scipy
- numpy
- matplotlib
- peakutils

This program is to be compiled by FBS into easily installable Windows program. Code is only for demonstration purpose. 
Ramanizator enables user to speed up analysis of Raman spectra in search of several factors of interests related to osteoarthritis, namely substitution of carbonate ions in locations of phosphate ions, crystallinity and mineralization of lattice.
